# 培训机构学分统计系统

这是一个为培训机构设计的学分统计系统，类似于商家会员积分系统。系统包含学员管理、积分统计和积分明细功能。

## 功能特性

### 学员管理
- 添加新学员
- 查看学员列表
- 显示学员当前学分

### 积分操作
- 为学员加分（奖励学分）
- 为学员扣分（扣除学分）
- 每次操作都有详细记录

### 积分明细
- 查看每个学员的完整积分历史
- 显示每次积分操作的时间、数量和说明
- 区分加分（绿色）和扣分（红色）

### 数据持久化
- 使用微信小程序本地存储保存数据
- 数据在关闭小程序后仍然保留

## 系统界面

### 主界面
- 学员列表区域：显示所有学员姓名和当前学分
- 添加学员区域：输入学员姓名添加新学员

### 操作界面
- 点击学员条目查看积分明细
- 点击"加分"按钮为学员增加学分
- 点击"扣分"按钮为学员扣除学分

### 弹窗界面
- 积分明细弹窗：显示学员所有积分操作记录
- 积分操作弹窗：输入积分数量和操作说明

## 使用方法

1. **添加学员**：在"添加新学员"区域输入学员姓名，点击"添加学员"按钮
2. **查看明细**：点击学员条目查看该学员的积分明细
3. **加分操作**：点击学员条目右侧的"加分"按钮，输入积分数量和说明
4. **扣分操作**：点击学员条目右侧的"扣分"按钮，输入积分数量和说明

## 技术特点

- 基于微信小程序开发
- 响应式设计，适配不同屏幕尺寸
- 现代化的UI设计
- 完整的错误处理和用户提示
- 数据本地存储，无需网络连接
